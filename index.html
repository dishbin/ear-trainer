welcome to the

semi-tone ear trainer

use this app to practice identifying notes and chords by ear

🎧 headphones recommended 🎧

press the 'begin training' button to begin a round

  • the default round settings play all 12 tones, from C4 to B4
  • the start button will turn green while the current note is being played
  • after each tone, pick which note you heard and it will automatically move to the next note in the round
  • the note you select will flash yellow to indicate that input was received
  • once the round ends, your score will be displayed along with a list of which notes you got correct and which you got incorrect

press the 'replay tone' button to hear the current tone or chord again

press the 'notes' button to manually select which notes to practice

  • notes with a checkmark will be included in the round
  • buttons for notes without a checkmark will not be displayed

use the 'chords' button to practice identifying chords

  • pressing the 'chords' button the first time will play only major chords
  • pressing the 'chords' button a second time will play only minor chords
  • pressing the 'chords' button a third time will play both types of chords
  • finally, pressing the 'chords' button a fourth time will disable chord mode

press the '∞' to enable infinite mode

  • upon pressing the '∞' button, the button's text will display a percent score
  • while in infinite mode, rounds will seamlessly restart
  • a running percentage score will be displayed on the infinite mode button
  • pressing the infinite mode while a score is displayed will disable infinite mode
  • changing the difficulty and/or enabling chord mode will disable infinte mode
  • you will have to start a new infinite round every time you adjust those settings

press the 'easy' button to adjust the dfiiculty

  • while in 'easy' mode, only tones between C4 and B4 will be played
  • chord mode on 'easy' will only play normal chords
  • playing in 'medium' mode will add half an octave above and below the default tones
  • chord mode on 'medium' will play normal chords as well as 1st inversion chords
  • playing in 'hard' mode will add a full octave above and below the default tones
  • chord mode on 'hard' will play normal chords, 1st inversion chords, and 2nd inversion chords
